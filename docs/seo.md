# SEO Strategy

The app generates conversion pages from `lib/conversions/data.ts` through `conversionPages()`.

## Page Pattern

- `/convert/{from}-to-{to}`
- Specific title and meta description
- Canonical URL
- FAQ schema
- Semantic headings and explanatory copy

## Expansion Plan

Add more units to existing categories to generate more pages automatically. Add new category objects for mining, pipe sizing, wire gauge, cooking and specialized agricultural conversions.

## Content Priorities

- Practical phrase matching: "bags of cement", "truckload of gravel", "liters per hectare"
- African regional terms and currencies
- Low-bandwidth pages with no unnecessary images
- Calculator pages for intent clusters like concrete, roofing, crop spacing and solar sizing
